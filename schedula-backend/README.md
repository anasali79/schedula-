# Schedula Backend

Simple NestJS-style boilerplate with a `GET /hello` endpoint.

## Setup

1. Install dependencies:

```bash
cd schedula-backend
npm install
```

2. Start the server in dev mode:

```bash
npm run start:dev
```

This will start the app on `http://localhost:3000`.

## API

- **GET `/hello`**
  - **Response**:

```json
{ "message": "Hello World" }
```
